# 智能射箭训练系统 - 界面原型

本文档提供系统主要页面的原型设计描述，帮助开发团队理解最终界面视觉效果。

## 登录页面原型

```
+-----------------------------------------------+
|                                               |
|            [系统Logo]                         |
|            智能射箭训练系统                    |
|                                               |
|    +-----------------------------------+      |
|    |                                   |      |
|    |  用户名/邮箱:                      |      |
|    |  +-----------------------------+  |      |
|    |  |                             |  |      |
|    |  +-----------------------------+  |      |
|    |                                   |      |
|    |  密码:                            |      |
|    |  +-----------------------------+  |      |
|    |  |                             |  |      |
|    |  +-----------------------------+  |      |
|    |                                   |      |
|    |  [✓] 记住我                       |      |
|    |                                   |      |
|    |  +-----------------------------+  |      |
|    |  |         登 录               |  |      |
|    |  +-----------------------------+  |      |
|    |                                   |      |
|    |  还没有账号? [注册]  [忘记密码?]    |      |
|    |                                   |      |
|    +-----------------------------------+      |
|                                               |
+-----------------------------------------------+
```

设计说明：
- 背景使用射箭运动相关的模糊照片，增强主题感
- 登录卡片使用半透明白色背景，确保清晰度
- 主色调按钮引导用户点击登录
- 错误提示将在输入框下方以红色文字显示

|                                               |

| 训练计划  |  |                | | 姿态分析    | |

| 个人档案  |  |   [视频播放]    | |            | |
| 设置      |  |                | | 动作分解    | |
|           |  |                | | [阶段分析]  | |
|           |  |                | |            | |
|           |  +----------------+ | 技术评分    | |
|           |  | [时间轴控制]    | | [得分图表]  | |
|           |  +----------------+ |            | |
|           |                     | AI建议     | |

设计说明：

- 视频下方时间轴上标记关键动作点
- 对比模式按钮允许用户切换到专业射手对比视图
- 分析数据与视频同步，滚动视频时更新数据

- 技能雷达图直观显示各项射箭技能水平

## 响应式设计说明

移动端设计将做如下调整：

1. **导航**
   - 侧边栏转为抽屉式菜单，通过汉堡按钮触发
   - 顶部导航栏简化，仅保留Logo和核心操作

2. **布局**
   - 卡片从多列布局改为单列垂直堆叠
   - 视频分析页面分析面板置于视频下方
   - 训练计划详情页面通过全屏弹出方式显示

3. **交互**
   - 增加滑动手势支持
   - 触摸目标尺寸增大，确保易于点击
   - 使用底部标签栏代替侧边栏进行主导航
